# Deferred Publication Shelf

This directory contains high-tier, cross-tier, or deferred-publication materials.

Files in this shelf are not automatically adopted into the current CYRUNE Free v0.1 public alpha claim. They require a separate publication decision before they can be treated as current public truth.

Some files in this shelf describe product-wide signed update or no-self-update policy. Those statements are deferred or product-wide material; they do not mean this Free v0.1 public alpha ships a signed updater, signing workflow, notarization workflow, or signed update channel.
